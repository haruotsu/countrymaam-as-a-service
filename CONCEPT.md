# Countrymaam as a Service (CMaaS)

> 焼き菓子を預けて、焼き菓子で支払う。  
> 不二家のカントリーマアム（※本プロジェクトは非公式のパロディです）をあたかも通貨・銀行資産のように扱う、冗談のような API & Webアプリ。

---

## コンセプト

「お金」を介さず、**カントリーマアム** そのものを資産として扱う銀行を作る。

- 預け入れ: カントリーマアムを金庫に入れる（＝残高が増える）
- 引き出し: 金庫からマアムを取り出す（＝残高が減る）
- 送金: 他のユーザーの金庫にマアムを送る
- 両替: バニラのマアムをチョコのマアムに換える（レート制）

現実の銀行のように**二重支払い**や**残高不足**を許さないよう、APIレベルでトランザクションで守る。パロディではあるが、お金ではなくお菓子で「ちゃんと動く銀行」を組み立てる、というのが本気の遊びポイント。

---

## 通貨単位

- **マアム (CM)** — カントリーマアム 1 枚 = 1 マアム
- 整数で扱う（割れたマアムは受け付けない。欠けは取引不能）

## フレーバー（= 通貨種）

v1 では以下の 3 フレーバーを公式サポート。口座は **ユーザー × フレーバー** で 1 つずつ持てる。

| フレーバー | 記号       | 基準レート | キャラ |
| ---------- | ---------- | ---------- | ------ |
| vanilla    | バニラ     | 1.0        | 定番・基軸通貨 |
| chocolate  | チョコ     | 1.2        | やや希少 |
| matcha     | 抹茶       | 1.5        | 季節限定プレミアム |

### 両替レート

`vanilla` を基準とした固定レート（v1）。両替時は次式で換算する。

```
amount_to = floor(amount_from × rate_from ÷ rate_to)
```

例:

- 10 バニラ → チョコ: `floor(10 × 1.0 / 1.2) = 8` チョコ
- 6 抹茶 → バニラ: `floor(6 × 1.5 / 1.0) = 9` バニラ

端数切り捨てで発生する差分はシステム側に帰属（手数料パロディ）。

## 主要オペレーション

| 操作         | 説明                                           | 方向           |
| ------------ | ---------------------------------------------- | -------------- |
| 口座開設     | ユーザーがフレーバーを選んで口座を作る         | —              |
| 預け入れ     | 口座にマアムを追加                             | `+amount`      |
| 引き出し     | 口座からマアムを取り出す                       | `-amount`      |
| 送金         | **同じフレーバー**の他口座へ送る               | `-amount / +amount` |
| 両替         | 同一ユーザー内で別フレーバー口座へ換算して移す | `-from / +to` |
| 残高照会     | 現在残高を取得                                 | —              |
| 取引履歴     | 当該口座の全取引を新しい順で取得               | —              |

## 不変条件（Invariants）

- 残高 `balance >= 0` — オーバードラフトなし
- 取引金額 `amount > 0` — 0 マアムの取引は存在しない
- 送金は **同一フレーバー間のみ**。違うフレーバーに送りたいときは両替を使う
- 送金・両替は **原子的** — 片側が失敗したら両口座とも変更されない
- 同一口座への自己送金は禁止

---

## 非目標（v1 では作らない）

- **利息** — 銀行っぽさの定番だが、今回のパロディでは不要とユーザーが判断
- 風化／賞味期限ロジック
- 為替レートの変動（固定）
- マルチテナント / 認可 / 認証（ユーザー ID をそのまま信じる）
- 監査ログの耐改竄性

これらは将来拡張の余地として残すが、v1 ではシンプルに保つ。

```

│   │   └── httpapi/        # chi ハンドラ
│   └── migrations/         # SQL マイグレーション
└── frontend/               # Next.js: ダッシュボード UI
```
